var B = Bakery;
primitives
B.draw.Point
new Point(x,y) create a point with the values of xy
x float value
y float value
.clone() create another point with have same xy values
.revertX() reverse x value
.revertY() reverse y value
.revert() reverse xy values
.add(point) add xy values of the point
.subtract() subtract xy values of the point
.distance(point) returns distance with the point
B.draw.Size
new Size(w,h) create a new Size object with width, height
width float value
height float value
.clone() clone itself
B.draw.Area
new Area(t,r,b,l) create a new Area object with top, right, bottom, left
top float value
right float value
bottom float value
left float value
width float value
height float value
size size object
.size(width, height) set width and height
.center() returns a Point object for center point
.clone() returns a clone
.expand(amount) expands it's length to all direction. center position will not change.
.shrink(amount) shrink it's length to all direction. center position will not change.
.contains(point) returns a bool value.
color
B.draw.color.RGBA
new RGBA(r,g,b,a) create new RGBA object
//r, g, b means red, blue, green, 0 ~ 255.
//a means alpha, 0.0 ~ 1.0
.toString() returns a string to represent color
B.draw.color.RGB RGBA
new RGB(r,g,b) create new RGB object
B.draw.color.Gray RGBA
new Gray(gray) create new Gray object.
new Gray(0) will be black.
new Gray(255) will be white.
B.draw.color.HSV
new HSV(h,s,v) create new HSV object
//h means hue, 0~360
//s, v means saturation, value of brgitness, 0~100
.toRGBA() convert to RGBA object
.toString() returns a string to represents color
.clone() clone it self
.addHue(h) add hue. 0 ~ 360
.addSaturation(s) add saturation. 0 ~ 100
.addBrightness(b) add brightness. 0 ~ 100
Animation
B.draw.Animation
new Animation(canvas) create new instance width canvas to draw.
.render(callback) Set callback for render. callback(ctx, CanvasSize, frameCount);
Don't forget to call .start() after .render()
.start() start animation
.stop() stop animation
.pause() pause animation
.remove() resume paused animation

point
size
area
fill flg to fill when draw. default true.
stroke flg to stroke when draw. default false.
fillColor instance of B.draw.color
strokeColor instance of B.draw.color
lineWidth line width for stroke
.toImg() return a canvas element.
B.draw.drawable.Rectangle
Drawable
new Rectangle() create a new Rectangle object.
you should set color, size, point before draw it.
.draw(ctx) draw the rectangle
B.draw.drawable.Circle
Drawable
new Circle() create a new Circle object.
you should set color, radius, point before draw it.
radius float value
.draw() draw the circle
B.draw.drawable.Ellipse
Drawable
new Ellipse() create a new Ellipse object.
you should set color, size, point before draw it.
.draw() draw the ellipse
B.draw.drawable.XMark
Drawable
new XMark() create new XMark object.
you should set color, radius, point before draw it.
.draw() draw the Xmark
B.draw.drawable.Star
Drawable
new Start() create a new Star object
you should it's color, outerRadius, innerRadius, point before draw it.
vertexCount number of vertex. default:5
outerRadius float value
innerRadius float value
.draw() draw the star
