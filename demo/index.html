About

What's bakery?
A JavaScript framework, which enables you a really simple, flexible and a bit perfunctory develop.
Baker provides:
  • Meta-object system for object-oriented programming
  • Various library to create client-rich web pages.
    • Baker.draw.js (coming soon!)
    • Baker.validator.js (coming soon!)
    • jQuery.baker.js (coming soon!)
Why bakery?
There are lots of JavaScript libraries which enables you object-orient programming. But most of them lose flexibility, which is the most wonderful feature of JavaScript, in exchange for strictness, stability, safety.
So I created this framework to use JavaScript as JavaScript, to make development flexible and a bit perfunctory and fun.
